Useful Linux Commands:

  • ls - List all files and directories in the current directory.
  • cd [directory] - Change directory to the specified one.
  • cd .. - Go out one directory
  • cat [file] - Display the content of a text file.
  • pwd - Display the current directory path.
  • rm [file] - Delete a specific file.
  • mv [source] [destination] - Move or rename a file.
  • cp [source] [destination] - Copy a file from one location to another.
  • touch [file] - Create a new empty file.
  • nano [file] - Open a text file in a basic terminal text editor.
