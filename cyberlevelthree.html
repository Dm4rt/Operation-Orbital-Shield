Mission Briefing - Network Defense

Your network is under threat from an adversary planning a coordinated cyberattack. Your task is to identify suspicious activity, analyze the logs, and block any malicious IPs that attempt to breach the system.

Mission Objectives:

  • 1. Use Nmap to scan for active threats.
  • 2. Analyze the network logs to detect suspicious activity.
  • 3. Set up firewall rules using iptables to block malicious IPs.

Step-by-Step Instructions:

Step 1: Scan the network for suspicious connections using Nmap:

nmap -sS 192.168.1.0/24

Step 2: Review the network logs for any suspicious connections:

grep 'Suspicious connection' /var/log/network.log

Step 3: Block any suspicious IP addresses using iptables:

iptables -A INPUT -s [Suspicious IP] -j DROP

This will prevent the attacker from accessing your network.

Once you've completed these steps, your mission will be complete!
