Mission Briefing - Detect & Disable Malware

R3dSh1ft has deployed a malicious script designed to retain access to this server. Your mission is to locate this backdoor, analyze its behavior, and disable it before further damage can occur.

Mission Objectives:

  • 1. Use the find command to search for hidden malicious scripts on the server.
  • 2. Analyze the contents of the script to determine its purpose.
  • 3. Disable or remove the script to prevent it from executing.

Step-by-Step Instructions:

Step 1: Search for malicious shell scripts that R3dSh1ft may have hidden:

find / -name ".r3dsh1ft*.sh" 2>/dev/null

This command will search the file system for any shell scripts related to R3dSh1ft.

Step 2: Analyze the contents of any discovered scripts:

cat .r3dsh1ft_backdoor.sh

Step 3: Disable or remove the script:

rm .r3dsh1ft_backdoor.sh

This will delete the backdoor script from the system.

Step 4: Confirm no other malicious scripts remain using find.
