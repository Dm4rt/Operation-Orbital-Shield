Mission Briefing - Password Cracking

The adversary, R3dSh1ft, is using weak credentials to protect key systems. Your mission is to collect possible passwords from various sources, build a custom password list, and use it to brute-force an SSH login.

Mission Objectives:

  • 1. Search through system files to gather potential passwords.
  • 2. Create a password list file with your findings.
  • 3. Use the Hydra tool to brute-force SSH access using your password list.
  • 4. Capture the correct password and proceed to the next mission.

Step-by-Step Instructions:

Step 1: Gather potential passwords from system files (e.g., `readme.txt`, `config.conf`, `users.txt`):

cat readme.txt cat users.txt

Search the system for other relevant files with find or grep commands.

Step 2: Create a password list file based on your findings:
(Hint: You have writing permissions in the tools folder)

nano passwordlistfile.txt

Step 3: Use Hydra to brute-force the SSH login:

hydra -l username -P passwordlistfile.txt [IP Address] -s 22 ssh

This command will attempt to log in to the SSH user using your custom password list.
