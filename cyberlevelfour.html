Mission Briefing - Incident Response

A critical system has been infected with malware. The adversary is attempting to spread the malware through your network, compromising key assets. Your mission is to detect the malware, isolate the infected systems, and remove the malicious code before it spreads further.

Mission Objectives:

  • 1. Use log analysis to detect the malware.
  • 2. Isolate the infected system using iptables.
  • 3. Run a malware scan to find and remove malicious files.
  • 4. Restore the system from backup and patch vulnerabilities.

Step-by-Step Instructions:

Step 1: Review the system logs for malware indicators.

grep 'malware' /var/log/system.log

Step 2: Isolate the infected system by blocking its IP address:

iptables -A INPUT -s [Infected IP] -j DROP

Step 3: Run a full malware scan to detect and remove infected files:

malwarescan /home/critical_files

Step 4: Restore the system from a backup and apply patches:

restore /backup/critical_files , patch -p1 /home/critical_files

Once you've completed these steps, proceed to secure the system!
